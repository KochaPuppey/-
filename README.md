1. Название: «Ввод корректного email адреса»
Шаги воспроизведения:
1)	Ввести валидный email
2)	Нажать кнопку «Проверить»
Ожидаемый результат: выводится alert с текстом «Прекрасный email-адрес»

2. Название: «Ввод email адреса на кириллице»
Шаги воспроизведения:
1)	Ввести email адрес на кириллице 
2)	Нажать кнопку «Проверить»
Ожидаемый результат: Выводится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email»

5. Название: «Перезагрузка страницы с введенным email адресом»

2)	Нажать кнопку «Перезагрузить страницу»
Ожидаемый результат: Форма с инпутом должно быть пустое, т.е очиститься 

6. Название: «Ввод вредоносного кода»
Шаги воспроизведения:
1)	Ввести вредоносный код
2)	Нажать кнопку проверить
Ожидаемый результат: Выводится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email». Вредоносный код не должен приводить к изменению DOM, иному негативному эффекту.
